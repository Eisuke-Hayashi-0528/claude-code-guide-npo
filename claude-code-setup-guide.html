Claude Codeって何？

ChatGPTでは、毎回「あなたはNPOの広報担当で…」と長い説明を入力していませんでしたか？ Claude Codeは、そういった「毎回説明する手間」を大幅に減らしてくれるツールです。

💡
一言で言うと

Claude Code ＝「Claudeをパソコン上で動かせるアプリ」です。
チャット画面ではなく、あなたのパソコンにインストールして使います。 ファイルを読み込んだり、文書を自動で作ったりと、より高度な使い方ができます。

ChatGPTとの違いは？

比較ポイント ChatGPT（ブラウザ版） Claude Code
使い方 ブラウザで開いて使う パソコンにインストール
毎回の説明 毎回入力が必要 CLAUDE.mdに一度書けばOK
ファイル操作 手動でコピー＆ペースト ファイルを直接読み書き
複数作業 1つずつ 複数の作業を連続して実行
カスタマイズ GPTsで設定（有料プラン） CLAUDE.mdで自由に設定
🎯
NPOでの活用イメージ

例えば「助成金の申請書を書いて」と伝えるだけで、
あなたの団体の理念・事業内容を把握した上で、申請書の下書きを自動生成することができます。 毎回「私たちは〇〇な団体で…」と説明する必要がなくなります！

2

なぜCLAUDE.mdが重要なの？

Claude Codeで最も大切なのが 「CLAUDE.md」 というファイルです。 これは、AIへの「永続的な説明書」です。

📖 CLAUDE.mdとは？

CLAUDE.mdとは、Claudeが作業を始める前に必ず読む設定ファイルです。
「私の団体はこういう団体です」「この言葉は使わないでください」「出力はいつも箇条書きで」
といったルールや背景情報を書いておくと、毎回の会話でそれが反映されます。

CLAUDE.mdがあると何が変わる？

  • 前
    CLAUDE.mdがない場合
    「私たちはNPO法人〇〇です。子どもの貧困問題に取り組んでいます。主な事業は… 対象は小学生から高校生で…ちなみに文章は丁寧語でお願いします…」 ←毎回これを書く必要がある 😰
  • 後
    CLAUDE.mdがある場合
    「ニュースレターの下書きを作って」だけで OK！🎉
    団体の情報・トーン・注意事項はCLAUDE.mdから自動で読み込まれます。
3

始める前に確認しよう

インストール前に、以下を確認してください。焦らず1つずつ確認してみましょう。

必要なもの

  • Claudeのアカウント（claude.ai）― 無料プランでも可（一部機能制限あり）
  • パソコン（Windows または Mac） ― スマートフォンでは動作しません
  • インターネット接続
  • 少しの好奇心！ ― 難しく考えなくて大丈夫です
⚠️
有料プランについて

Claude Code はClaudeの有料プラン（Pro: 月額$20〜）または API利用が必要です。
本格的な業務利用には有料プランをおすすめします。まず無料で試してから検討してもOKです。

難易度の目安

作業難易度所要時間
Claude Codeのインストール ★★☆ やや難しい 10〜20分
CLAUDE.mdの作成 ★☆☆ かんたん 10〜15分
日常利用に慣れる ★☆☆ かんたん 数日〜1週間
4

Claude Codeをインストールする

Claude CodeはAnthropicが公式に提供しているツールです。 難しそうに聞こえますが、手順通りに進めれば大丈夫です。一緒に進みましょう。

方法① デスクトップアプリ（おすすめ・最も簡単）

  • 1
    Claudeのデスクトップアプリをダウンロードする
    claude.ai/download にアクセスして、Mac版またはWindows版をダウンロードします。
  • 2
    アプリをインストールする
    ダウンロードしたファイルを開き、通常のアプリと同様にインストールします。
    Macの場合：.dmgファイルを開いて Applications にドラッグ
    Windowsの場合：.exeファイルを実行して指示に従う
  • 3
    Claudeアカウントでログインする
    アプリを起動し、claude.aiのアカウントでログインします。
    「Claude Codeを使用する」または「Code」タブを選択してください。
  • 4
    作業フォルダを選ぶ
    最初に「どのフォルダで作業しますか？」と聞かれます。
    例えば「NPO業務」フォルダや「デスクトップ」を選んでOKです。
    このフォルダの中にCLAUDE.mdを作成します。
🖥️
「作業フォルダ」って何？

Claude Codeは「フォルダ単位」で動作します。 例えば「NPO仕事」というフォルダを作り、その中で使うと、 そのフォルダ内のファイルをClaudeが参照・作成できるようになります。 まずはデスクトップに「claude-test」フォルダを作って試してみましょう。

方法② VSCode拡張機能（少し慣れてきたら）

Visual Studio Code（VSCode）を使っている方は、拡張機能からClaude Codeをインストールする方法もあります。 まずはデスクトップアプリで慣れてから試してみてください。

✅
インストール完了の確認

アプリを起動して、チャット画面が表示されたらインストール成功です！
最初に「こんにちは！何か手伝えることはありますか？」と聞いてみましょう。

5

CLAUDE.mdを作ってみよう

いよいよメインの作業です！CLAUDE.mdを作ることで、Claudeがあなたの団体を「覚えて」くれます。

CLAUDE.mdファイルの作り方

  • 1
    作業フォルダを開く
    先ほど選んだ作業フォルダ（例：「NPO業務」フォルダ）をエクスプローラーまたはFinderで開きます。
  • 2
    「CLAUDE.md」という名前でファイルを作る
    メモ帳（Windows）またはテキストエディット（Mac）を開き、
    名前を 「CLAUDE.md」（大文字・拡張子含む）として保存します。
    ⚠ 「CLAUDE.txt」ではなく「CLAUDE.md」にしてください
  • 3
    内容を書く
    次のセクションのテンプレートを参考に、あなたの団体の情報を書き込みます。
    難しく考えず、「AIへのメモ書き」くらいの気持ちで書いてOKです。
  • 4
    Claude Codeから確認する
    Claude Codeアプリで同じフォルダを開き、「CLAUDE.mdを読んで、私の団体について教えて」と入力してみましょう。 正しく読み込まれていれば、団体の情報を答えてくれます。
⚠️
ファイル名の注意点

ファイル名は必ず CLAUDE.md（すべて大文字）にしてください。
小文字の「claude.md」でも動作しますが、大文字が推奨されています。
また、フォルダの直下（一番上の階層）に置くことが重要です。

CLAUDE.mdに書くべき4つの内容

🏢
① 団体プロフィール

団体名・設立年・ミッション・主な事業・対象者

✍️
② 文章のルール

使う言葉・使わない言葉・文体（丁寧語か？）・文字数の目安

⚡
③ よくある依頼

報告書・ニュースレター・SNS投稿など、よく作る文書の型

🚫
④ 禁止事項・注意点

個人情報の扱い・書いてはいけないこと・確認が必要な事項

6

NPO向けCLAUDE.mdテンプレート

以下のテンプレートをコピーして、【　】の部分をあなたの団体の情報に書き換えてください。 書き換えたものをCLAUDE.mdとして保存するだけでOKです。

📄 CLAUDE.md テンプレート（コピーしてお使いください）
# 私の団体について（AIへの説明）

## 基本情報
- 団体名：【NPO法人〇〇〇〇】
- 設立年：【〇〇〇〇年】
- 所在地：【〇〇県〇〇市】
- スタッフ数：【常勤〇名、非常勤〇名、ボランティア〇名程度】

## ミッション・活動内容
【私たちは〇〇を目指し、〇〇な活動をしています。
主な対象者は〇〇で、〇〇などの事業を行っています。】

## よく使う言葉・キーワード
- 【例：「伴走支援」「当事者目線」「地域共生」など】

## 文章のルール
- 文体：【丁寧語（です・ます調）】
- 語り口：【温かみがあり、専門用語は避けてわかりやすく】
- 避けるべき表現：【例：「弱者」「問題児」など否定的なレッテル表現】
- 文字数の目安：【SNS投稿は200文字以内、報告書は400〜800字程度】

## よく作る文書の種類
- 助成金申請書の下書き
- 月次ニュースレター（支援者向け）
- 活動報告書
- SNS投稿文（Facebook・Instagram）
- 議事録の整理

## 注意事項（必ずこれを守ってください）
- 個人情報（利用者の名前・住所等）は文書に含めない
- 金額・数字は必ず私が確認するまで確定しない
- 助成金名・法人名は正式名称を使う
- 不確かな情報は「要確認」と明示する

## 担当者について
- 私の役職：【事務局長 / 〇〇担当スタッフ】
- 私の得意・不得意：【文書作成は得意、数字の整理が苦手】
- AIに期待すること：【下書きの作成・アイデア出し・文書の校正】

書き換えのヒント

💬
うまく書けない場合は、Claudeに聞いてみよう！

Claudeに「私の団体はこんな団体です：〇〇〇〇。これをCLAUDE.md用に整理してください」と伝えると、 CLAUDE.mdの文章を自動で作ってくれます。最初から完璧に書く必要はありません。

記入例（子ども支援系NPOの場合）

📝 記入例（参考）
# 私の団体について（AIへの説明）

## 基本情報
- 団体名：NPO法人こどもサポートセンター〇〇
- 設立年：2015年
- 所在地：東京都〇〇区
- スタッフ数：常勤3名、非常勤5名、ボランティア30名程度

## ミッション・活動内容
すべての子どもが安心して育てる地域社会を目指しています。
主な対象は経済的困難を抱える家庭の子ども（小〜高校生）で、
学習支援・食事支援・居場所づくりを行っています。

## よく使う言葉・キーワード
- 「子どもの貧困」「居場所」「食育」「無料塾」「学習支援」

## 文章のルール
- 文体：です・ます調
- 語り口：温かく、保護者や地域の方に伝わるわかりやすい言葉で
- 避けるべき表現：「かわいそう」「問題家庭」などのレッテル表現
- 文字数：SNSは150〜200字、ニュースレターは400〜600字

## 注意事項
- 子どもや保護者の個人情報は一切含めない
- 数値・統計は出典を明記、または「要確認」と記載
7

うまく使うためのコツ

コツ① まず試して、少しずつ育てる

CLAUDE.mdは最初から完璧でなくてOKです。使いながら「ここが足りなかった」と気づいたら追記していきましょう。 AIと一緒に成長させていくイメージです。

🌱
改善のサイクル

使う → 「なんか違う」と感じる → CLAUDE.mdに追記・修正 → また使う
このサイクルを繰り返すことで、どんどん使いやすくなります。

コツ② 具体的な指示を出す

あいまいな指示 ❌具体的な指示 ✅
「文章を書いて」 「500字以内で、Facebookに投稿する夏祭りの告知文を書いて」
「報告書をまとめて」 「添付の議事録から、箇条書きで3点の決定事項を抜き出して」
「助成金の文章を直して」 「この文章をより具体的で説得力のある表現に修正して。対象は助成財団の審査員」

コツ③ 絶対に守るべきこと（情報セキュリティ）

🔒
個人情報は入力しない！

利用者・支援対象者の氏名・住所・電話番号・健康情報などをClaude Codeに入力しないでください。
AIのサービスはデータが外部に送信されます。個人情報保護の観点から、必ず匿名化・一般化してから入力しましょう。

❌「田中花子さん（〇〇区在住、15歳）の支援記録は…」
✅「支援対象者（15歳女子）のケース記録は…」

📋
AIの出力は必ず確認・修正する

Claudeが作った文章は「下書き」です。数字・固有名詞・事実関係は必ず人間が確認してください。 AIは誤った情報を自信満々に答えることがあります（ハルシネーションと呼ばれます）。 最終責任は常に人間が持ちましょう。

コツ④ 便利な使い方アイデア（NPO向け）

📝 議事録の整理

メモやボイスメモの文字起こしをペーストして「決定事項・次のアクション・担当者を整理して」

📊 助成金申請書の下書き

「〇〇財団の申請書の『活動の必要性』欄（400字以内）を書いて」と依頼

📣 SNS・ニュースレター

「先月の活動レポートをもとに、Facebookの投稿文を3案作って」

📧 メール下書き

「行政担当者への事業報告メールを書いて。敬語・丁寧な表現で」
